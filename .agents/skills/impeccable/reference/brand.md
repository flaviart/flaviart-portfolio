# Reference: Brand Interfaces

## Characteristics

Brand interfaces (portfólio, landing page, campanha, editorial) priorizam **impressão, emoção e clareza de mensagem** sobre funcionalidade pura.

## Key Differences from Product

| Aspect | Product | Brand |
|--------|---------|-------|
| Primary goal | Task completion | Emotional connection / conversion |
| Navigation | Persistent, dense | Minimal, contextual |
| Content density | Higher, information-rich | Lower, generous whitespace |
| Visual risk | Lower, consistency wins | Higher, personality wins |
| CTA focus | Action-oriented (save, submit) | Contact-oriented (hire, view, email) |

## Design Priorities for Brand

1. **Hero impact**: Os primeiros 100vh são críticos. Título, subtítulo e imagem devem comunicar identidade imediatamente.
2. **Whitespace como elemento de design**: Não é ausência de conteúdo, é respiração intencional.
3. **Tipografia como identidade**: A escolha de fontes pesa mais que em produtos funcionais.
4. **Narrativa visual**: A página deve contar uma história ao rolar. Cada seção é um capítulo.
5. **Social proof**: Depoimentos, clientes, ou reconhecimentos têm peso maior para validar credibilidade.
6. **Simplicidade na navegação**: Links diretos: Work, About, Contact. Sem dropdowns complexos.

## Common Pitfalls

- Layout simétrico demais (parece template).
- Texto genérico ("Bem-vindo ao meu site").
- Fotos de stock sem personalização.
- Cores que não refletem a personalidade do criador.
- Ausência de CTA clara no final da página.

## Polishing Checklist for Brand

- [ ] Hero tem hierarquia clara (título > subtítulo > CTA > imagem)?
- [ ] Espaçamento entre seções é generoso e consistente?
- [ ] Tipografia transmite personalidade sem sacrificar legibilidade?
- [ ] Cores são consistentes e têm significado?
- [ ] Imagens são de alta qualidade e carregam rápido?
- [ ] CTA de contato é fácil de encontrar?
- [ ] Footer tem informações essenciais sem excesso?
- [ ] Animações são sutis e não distraem?
- [ ] Mobile preserva o impacto visual do desktop?
